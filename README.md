# Detecting-Pneumonia-from-X-ray-Images-Deep-Learning
### Description
Prediction of PNEUMONIA based on Chest X-ray Images using Convolutional Neural Network and tensorflow - Deep Learning approach.

### Python libraries used :

1. Tensorflow
2. Keras
3. Scikit-learn
4. Numpy
5. Matplotlib
6. Seaborn
